The preoperative model uses variables available before surgery, whereas the extended perioperative model additionally incorporates operative and catheter-management variables available before first catheter removal.

The calculator uses averaged ridge penalized logistic regression coefficients across 20 multiply imputed datasets. Continuous variables are entered in their original units. Binary variables are coded as Yes = 1 and No = 0. Procedure type is coded as Partial colpocleisis = 1 and Modified colpocleisis = 0. Ridge coefficients are prediction weights and should not be interpreted as independent risk-factor estimates.
